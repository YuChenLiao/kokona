# Blue_Archive_ZhinBot

碧蓝档案总力攻略图、角色查询插件, 适用于由 zhin 框架所开发的Bot

实现非常粗糙且简陋，请各位巨佬嘴下留情

后续功能仍在开发中，将来吧反正，咕咕咕

## 安装方法

1、在 zhin 的插件目录 plugins 下clone本项目并安装axios

2、在 zhin 项目根目录下使用npm安装
